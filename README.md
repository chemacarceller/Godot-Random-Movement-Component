# Godot-Random-Movement_Component

This is the development of a basic random movement component written in C++ designed to be attached to a Node3D in Godot

This component has been tested in https://jocarpe.itch.io/third-person-character-demo

This is a demo in continuous development to test all the components developed and in the process of future development. 

A detailed explanation of how the demo works is available on the itch.io page indicated. 

Feel free to check it out in either of its two versions, Windows or Linux.

The code for this demo is available at : [Third Person Character Demo](https://github.com/chemacarceller/Godot-Third-Person-Character-Demo) 

The same demo is available for First Person Character in https://jocarpe.itch.io/godots-first-person

This repository contains the component's .cpp and -h files, as well as the register_types -cpp and .h files necessary for compiling the module.

If you only want to use the module without modifying the code, the .so and .dll files for both denug and pararelease are provided. These must be placed in the res://main/bin directory of your project, as specified in the .gdextension file that is also provided.
